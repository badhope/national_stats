## 程序列表

### 1. main.py - 程序主入口
   用途：程序启动入口
   功能：协调各模块运行

### 2. analyzer.py - 数据分析器
   用途：宏观经济数据分析
   功能：数据处理、指标计算、统计分析

### 3. visualizer.py - 数据可视化
   用途：生成数据图表
   功能：绘制折线图、柱状图、饼图等

### 4. predictor.py - 趋势预测
   用途：经济趋势预测
   功能：基于历史数据进行趋势分析

### 5. reporter.py - 报告生成
   用途：生成分析报告
   功能：输出文本报告、摘要信息

### 6. requirements.txt - 依赖列表
   用途：程序运行所需Python包

## 运行方式

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行主程序
python main.py

# 3. 查看帮助
python main.py --help
```

## 输出说明

程序运行后会在以下目录生成输出：
- output/          - 分析结果输出
- logs/            - 运行日志
- cache/           - 缓存数据

## 注意事项

⚠️ 本程序为独立运行的功能程序：
- 不依赖静态展示网页
- 不引用 docs-presentation/ 目录
- 需要 Python 3.8+ 环境
- 需要网络连接获取数据（可选）
